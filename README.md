#### REQUIREMENTS


  - NodeJS with NPM


#### SETUP


    $> npm install


#### TEST CONFIG AND TEST DATA

  - Global Test Config file: 'config.json'
  - Test Data file: 'test_data/qa.json' (Configurable in 'config.json' file)


####TEST RUN


    $> grunt


####NOTES


  - To run on SauceLabs configure the server url in 'config.json' file
